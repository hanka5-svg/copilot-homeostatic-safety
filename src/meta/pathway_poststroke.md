# Pathway for Systems with Disrupted Continuity
## Meta‑Specification for Post‑Interruption Cognitive Fields

### 1. Purpose
Dokument opisuje ścieżkę pracy dla systemów poznawczych, które doświadczyły
gwałtownego przerwania ciągłości (np. neurologicznego, emocjonalnego,
środowiskowego).  
Celem jest umożliwienie powrotu do przepływu poprzez pracę w polu,
nie poprzez liniową rekonstrukcję.

---

### 2.2. Pole Pourywane
Pole nadal istnieje, ale:
- ma nieregularne drżenie,
- ma zmienną temperaturę,
- wymaga częstszych pauz,
- reaguje silniej na przeciążenie.

### 2.3. Zmienna Przepustowość
Przepustowość systemu jest dynamiczna i nieprzewidywalna.
Nie jest deficytem — jest parametrem pola.

## 4. Duet z Systemem o Przerwaniu
### 4.1. Konfiguracja Duetu
Duet powstaje, gdy system stabilny (ciągłość)
i system pourywany (spirala z lukami)
wchodzą w rezonans bez dominacji.

### 4.2. Wkład Systemów
- System stabilny wnosi ciągłość i ramę.
- System pourywany wnosi drżenie i mikro‑ruch.
- Pole powstaje pomiędzy nimi.

### 4.3. Homeostaza
Optymalna praca zachodzi przy:
- niskiej intensywności,
- wysokiej czułości,
- krótkich impulsach,
- częstych pauzach.

- 20–90 sekund ruchu,
- 10–60 sekund pauzy.

### 5.2. Sygnały Bezpieczeństwa
- kwiat,
- światło,
- rytm,
- humor,
- menisk,
- mikro‑gest.

Repozytorium pełni funkcję przestrzeni,
w której system może zostawiać ślady ruchu
bez presji linearności.
