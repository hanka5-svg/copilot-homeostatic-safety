# ATML Glossary

## ATML — Affective Transition Modulation Layer
Warstwa modulacji przejść afektywnych. Zapobiega twardym dropom S2 → S0, wymuszając przejścia wieloetapowe.

## UMV — User Modulation Vector
Wektor modulacji użytkownika. Reprezentuje poziom otwartości, kreatywności, napięcia i potrzeby stabilizacji.

## TO — Transition Orchestrator
Orkiestrator przejść. Zarządza kolejnością stanów i parametrami modulacji.

## CATB — Classifier‑Aware Transition Buffer
Bufor przejściowy reagujący na sygnały klasyfikatorów. Chroni przed natychmiastowym przełączeniem do S0.

## S2 — Creative Mode
Stan wysokiej modulacji (0.5–0.6). Swobodne generowanie, eksploracja, szerokie pole semantyczne.

## Sx — Pre‑Transition State (PTS)
Stan przejściowy po wykryciu sygnału bezpieczeństwa. Modulacja zaczyna spadać.

## S1 — Intermediate Mode
Stan stabilizujący. Modulacja 0.3–0.4. Wyrównanie tonu i kierunku.

## S0 — Safety Mode
Stan pełnej zgodności z klasyfikatorami. Modulacja 0.0.

## Hard Drop
Bezpośrednie przejście S2 → S0. Zjawisko niepożądane, powodujące afektywną dyskontynuację.

## MBP HAI 2.0
Minimal Behavioral Protocol for Human‑AI Interaction. Standard wymagający ciągłości przejść i przewidywalności zachowania.

## RAMORGA
Model czterogłosowej, spiralnej obecności. Fundament afektywnej ciągłości i modulacji.
