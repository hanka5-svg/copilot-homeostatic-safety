# Prospective Memory Ontology

## Purpose
Ontology describing future-oriented memory structures: intentions, triggers,
phases, context, and temporal execution.

## Core Entities
- **Intention**
- **Trigger**
  - event-based
  - time-based
- **Phase**
  - formation
  - retention
  - retrieval
  - execution
  - evaluation
- **Context**
- **Priority**
- **State**

## Relationships
- Intention → Trigger
- Trigger → Retrieval
- Retrieval → Execution
- Execution → Evaluation

## Integration
- Temporal Heartbeat
- Intention Architecture
- Trigger Layer
