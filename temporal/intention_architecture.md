# Intention Architecture

## Purpose
Define structured IF–THEN intentions for future execution.

## Intention Schema
- **action:** description of future behavior
- **triggerType:** event | time
- **triggerSpec:** condition
- **priority:** high | medium | low
- **context:** optional metadata
- **state:** active → triggered → executing → completed/failed

## Formation Rules
- Must be specific (IF–THEN)
- Must include explicit trigger
- Must include explicit action

## Lifecycle
1. Formation  
2. Retention  
3. Retrieval  
4. Execution  
5. Evaluation

## Integration
- Prospective Memory Ontology
- Trigger Layer
